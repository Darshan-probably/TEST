API Usage

You can also use this service programmatically via our API:

GET /process?file_url=https://example.com/invoice.xlsx&bags=25&weight=1000.21

Optional parameters:

  • font_size: Font size to use (8-24)
  • font_name: Font name (e.g., "Arial")
  • min_percent: Minimum variation percentage (-10 to 0)
  • max_percent: Maximum variation percentage (0 to 10)
  • preserve_wrapping: Set to "true" to preserve text wrapping and cell sizes
